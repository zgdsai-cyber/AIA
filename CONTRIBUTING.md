# Contributing to AIA

شكراً لاهتمامك بالمساهمة في مشروع AIA! نرحب بمساهماتك وأفكارك.

## كيفية المساهمة

### 1. الإبلاغ عن الأخطاء
إذا وجدت خطأ، يرجى فتح Issue جديد مع:
- وصف واضح للمشكلة
- خطوات إعادة الإنتاج
- الحد الأدنى من الكود المتكرر
- نظام التشغيل والمتصفح

### 2. الميزات الجديدة
إذا كان لديك فكرة ميزة جديدة:
- اقتح Issue أولاً لمناقشة الفكرة
- انتظر الموافقة من الفريق
- أنشئ Pull Request مع التطبيق

### 3. التحسينات
نقبل تحسينات الأداء والتحسينات الأخرى.

## سير العمل

1. **Fork** المستودع
```bash
git clone https://github.com/yourusername/AIA.git
cd AIA
```

2. **أنشئ فرع** جديد
```bash
git checkout -b feature/your-feature
```

3. **اعمل على التغييرات**
```bash
# قم بإجراء التغييرات
npm run dev  # اختبر محلياً
```

4. **Commit التغييرات**
```bash
git add .
git commit -m "Add your feature description"
```

5. **Push إلى GitHub**
```bash
git push origin feature/your-feature
```

6. **فتح Pull Request**
- اشرح التغييرات
- أرفق الصور إن أمكن
- اذكر أي Issues ذات صلة

## معايير الكود

- استخدم TypeScript
- اتبع نمط الكود الموجود
- أضف comments حيث لزم الأمر
- اختبر المتصفحات المختلفة
- تأكد من التجاوب (Responsiveness)

## آداب التواصل

- كن محترماً ولطيفاً
- استقبل النقد بصدر رحب
- ركز على الكود وليس على الشخص
- اتبع قيم الجماعة المفتوحة

شكراً لمساهمتك! 🙏
