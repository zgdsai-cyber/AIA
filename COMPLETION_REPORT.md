## 🚀 كيفية الاستخدام

### البدء السريع:

```bash
# 1. الذهاب إلى المشروع
cd /workspaces/AIA

# 2. تثبيت الحزم
npm install

# 3. تشغيل التطبيق
npm run dev

# 4. فتح المتصفح
# http://localhost:3000
```

### استخدام النموذج:
1. انتقل إلى الصفحة الرئيسية
2. ملأ البيانات الأربعة (4 خطوات)
3. اضغط "إنشاء المتجر"
4. معاينة التصميم النهائي
