## مكونات الواجهة الأفساسية

### 1. الأزرار (Buttons)
```
الأنواع المتاحة:
- Primary:   خلفية بالون الأساسي، نص أبيض
- Secondary: خلفية بالون الثانوي، نص من اللون الأساسي
- Outline:   حد من اللون الأساسي، نص من اللون الأساسي
- Ghost:     بدون خلفية، نص من اللون الأساسي

الأحجام:
- SM:  px-3 py-2 text-sm
- MD:  px-6 py-3 text-base
- LG:  px-8 py-4 text-lg

الحالات:
- Default:  عادي
- Hover:    تغيير الشفافية
- Active:   ظل إضافي
- Disabled: شفافية 50%
```

### 2. الحقول (Inputs)
```
التصميم:
- حد 2px بلون الإضاءة
- زوايا مستديرة (border-radius: 8px)
- تركيز: تغير اللون للون الثانوي
- خطأ: تغير اللون للأحمر

المتغيرات:
- Text input
- Email input
- Number input
- Textarea
- Select dropdown
```

### 3. البطاقات (Cards)
```
التصميم:
- خلفية بيضاء
- حد خفيف من الإضاءة
- زوايا مستديرة 12px
- ظل ناعم على الافتراضي
- ظل أقوى عند التمرير

التباعد:
- Padding: 24px من الداخل
- Margin: حسب السياق
```

### 4. النماذج (Forms)
```
التخطيط:
- عنوان النموذج في الأعلى
- تسمية الحقول بوضوح
- رسائل الخطأ بالأحمر
- زرار الإرسال بارز

التباعد:
- بين الحقول: 16px
- من الأعلى: 24px
```

## حالات التفاعل

### Hover States:
```
الأزرار:       شفافية أعلى أو ظل أقوى
البطاقات:     ظل أقوى
الروابط:      تغيير اللون
الحقول:       يبقى كما هو
```

### Active States:
```
الأزرار:       لون أغمق أو ظل إضافي
البطاقات:     تكبير طفيف
الروابط:      لون مختلف + underline
```

### Disabled States:
```
كل العناصر:    شفافية 50% + cursor: not-allowed
```

### Focus States:
```
الحقول:        تغيير لون الحد للون الثانوي
الأزرار:       حد ملون
```
